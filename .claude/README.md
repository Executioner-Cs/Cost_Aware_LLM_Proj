# .claude/

This directory is git-ignored. It holds local Claude Code instructions for this
repository, kept on your machine only.

* Put your own local Claude Code operating notes here, or in `CLAUDE.local.md` at
  the repo root. They stay local.
* Do not commit internal prompts, agents, skills, logs, cache, or any secrets.
* Only this `README.md` is tracked; everything else under `.claude/` is ignored.

For a starting point, copy `docs/development/CLAUDE_LOCAL_TEMPLATE.md` to
`CLAUDE.local.md` and edit it locally.
